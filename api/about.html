Esse é uma API em desenvolvimento, criada por eumsmo sem intuito profissional. Essa API foi feita tendo como publico alvo eu e qualquer outro desenvolvedor web, ou jogador de Don't Starve que tá atoa.

Uso livre, mas se começar dar ruim eu vou tirar do ar, então favor colaborar. Para mais informações entre em contato comigo pelo Github ou Twitter(se você quiser muito)!

  • /item : devolve um vetor com todos os itens e suas respectivas informações
  • /item/_NOME_DO_ITEM_ : devolve um objeto com as informações de um item especifico
  • /itemlist : devolve um vetor com o nome de todos os items craftaveis
  • /resourcelist : devolve um vetor com o nome de todos os recursos utilizaveis para craftar
