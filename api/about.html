Sobre

Esse é uma API em desenvolvimento, criada por eumsmo sem intuito profissional. Essa API foi feita tendo como publico alvo eu e qualquer outro desenvolvedor web, ou jogador de Don't Starve que tá atoa.

Uso livre, mas se começar dar ruim eu vou tirar do ar, então favor colaborar. Para mais informações entre em contato comigo pelo Github ou Twitter(se você quiser muito)!

  • /item : devolve um objeto com todos os itens e suas respectivas informações
    • /item/(nome) : devolve um objeto com as informações de um item especifico
    • /item/(nome)/(propriedade) : devolve um JSON com uma propriedade de um item especifico
  • /resource : devolve um objeto com todos os recursos utilizaveis para craftar e todos os itens que tal recurso pode craftar
    • /resource/(nome) : devolve um vetor com todos os itens que tal recurso pode craftar
  • /image[.png] : devolve uma spritesheet(.png) com todos os itens
    • /image/(nome)[.png] : devolve a imagem(.png) relativa ao item
  • /list :
    • /list/resource : devolve um vetor com o nome de todos os recursos utilizaveis para craftar
    • /list/item : devolve um vetor com o nome de todos os items craftaveis
    • /list/image : devolve um vetor com o nome de todas as imagens da API
